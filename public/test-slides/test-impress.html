✦ SLIDE 02

3D 위치 이동

Impress.js는 각 슬라이드(step)에 data-x, data-y, data-z 속성을 지정하여 3차원 공간에 자유롭게 배치합니다.
카메라가 각 요소로 이동하면서 프레젠테이션이 진행됩니다.

X →
수평 이동
(양수 = 오른쪽)
Y ↓
수직 이동
(양수 = 아래)
Z ◎
깊이 이동
(양수 = 앞으로)
✦ SLIDE 03

회전과 스케일

  • data-rotate Z축 기준 2D 회전 (도°)
  • data-rotate-x X축 기준 3D 회전 — 앞뒤로 기울기
  • data-rotate-y Y축 기준 3D 회전 — 좌우로 기울기
  • data-rotate-z Z축 기준 회전 (rotate와 동일)
✓
data-rotate로 Z축 회전 이해했음
✓
data-rotate-x / Y로 3D 기울기 이해했음
✓
data-scale로 확대·축소 이해했음
✦ SLIDE 04

깊이감 (Z축)

data-z를 음수로 설정하면 슬라이드가 화면 뒤쪽으로 물러나 있는 것처럼 보입니다. data-rotate-x와 결합하면 강한 3D 원근감을 표현할 수 있습니다.

<div class="step" data-x="1500" data-y="1500" data-z="-500" data-rotate-x="45" data-scale="1">
0
Z축 깊이 체감 카운터

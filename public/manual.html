HTML 편집기 사용 설명서비개발자를 위한 시각적 HTML 편집 도구

HTML 편집기는 코드를 몰라도 HTML 파일을 시각적으로 수정할 수 있는 도구입니다. 요소를 클릭하여 선택하고, 왼쪽 패널에서 색상·크기·여백 등을 조정하면 즉시 미리보기에 반영됩니다.

개요

🎯

시각적 편집

요소를 클릭하면 해당 속성을 패널에서 즉시 편집할 수 있습니다.

⚡

실시간 미리보기

변경 사항이 바로 미리보기에 반영됩니다. 코드 뷰로 전환하여 직접 수정도 가능합니다.

↩️

실행 취소/다시 실행

최대 50단계 이력 관리로 실수를 쉽게 되돌릴 수 있습니다.

📸

이미지 캡쳐

페이지 섹션을 자동 감지하여 고해상도 이미지로 내보낼 수 있습니다.

빠른 시작

1
파일 열기 — 왼쪽 패널의 업로드 영역에 .html 파일을 드래그하거나 클릭하여 선택합니다.
2
요소 선택 — 미리보기에서 편집할 요소를 클릭합니다. 왼쪽 패널에 해당 속성이 표시됩니다.
3
속성 편집 — 색상, 크기, 텍스트 등 원하는 속성을 조정합니다. 변경 사항이 즉시 미리보기에 반영됩니다.
4
저장 — 상단 저장 버튼(Ctrl+S)을 클릭하여 수정된 HTML 파일을 다운로드합니다.

파일 열기

왼쪽 패널 상단의 업로드 영역을 통해 파일을 불러옵니다.

  • 드래그 앤 드롭 — .html, .htm 파일을 영역 위로 드래그합니다.
  • 클릭하여 찾아보기 — 업로드 영역 또는 파일 변경 버튼을 클릭합니다.
  • 붙여넣기 — 파일을 클립보드에 복사한 뒤 붙여넣습니다.
파일 크기는 최대 10MB입니다. 이미 파일이 열려 있을 때 새 파일을 불러오면 기존 편집 내용이 초기화됩니다.

뷰 모드 전환

오른쪽 패널 상단의 프리뷰 / 코드 버튼으로 뷰를 전환합니다.

모드설명
프리뷰 HTML이 렌더링된 실제 화면. 요소를 클릭하여 선택 및 편집. 슬라이드덱·인터랙티브 요소도 동작.
코드 Monaco 코드 에디터(VS Code 방식). HTML 구문 강조, 자동 완성. 코드 수정 후 프리뷰로 전환 시 자동 동기화.
프리뷰에서 요소를 선택한 뒤 코드 뷰로 전환하면 해당 요소가 있는 코드 라인으로 자동 스크롤되고 하이라이트됩니다.

클래스 일괄 적용

같은 클래스를 가진 모든 요소에 속성 변경을 한 번에 적용하는 기능입니다.

1
클래스가 있는 요소를 선택합니다.
2
왼쪽 패널의 "같은 클래스에 모두 적용" 체크박스를 활성화합니다.
3
드롭다운에서 편집할 클래스를 선택합니다.
4
속성을 변경하면 해당 클래스를 가진 모든 요소에 동시 적용됩니다.

실행 취소 / 다시 실행

상단 툴바 또는 단축키로 변경 이력을 탐색합니다. 최대 50단계까지 이력이 유지됩니다.

버튼단축키동작
실행 취소Ctrl+Z직전 변경을 되돌립니다.
다시 실행Ctrl+Y 또는 Ctrl+Shift+Z취소한 변경을 다시 적용합니다.
원본 복원—파일을 처음 열었을 때 상태로 완전히 되돌립니다.
슬라이드덱 HTML에서 실행 취소/다시 실행 시, 변경했던 슬라이드 페이지로 자동 이동합니다.

이미지 캡쳐

페이지를 섹션별로 나눠 고해상도 이미지로 내보냅니다. 왼쪽 패널 하단 캡쳐 영역에서 사용합니다.

1
형식(JPG / PNG)과 해상도(1x / 2x / 3x)를 선택합니다.
2
섹션 감지 버튼을 클릭합니다. 페이지의 구분 섹션이 자동으로 감지됩니다.
3
감지된 섹션 목록과 높이를 확인합니다. 원하지 않을 경우 초기화 후 다시 감지합니다.
4
캡쳐 버튼을 클릭합니다. 진행바가 표시되고 완료되면 ZIP 파일이 다운로드됩니다.
캡쳐는 프리뷰 모드에서만 작동합니다. 코드 뷰에서는 먼저 프리뷰로 전환하세요.

저장

상단 저장 버튼 또는 Ctrl+S를 누르면 수정된 HTML 파일이 컴퓨터에 다운로드됩니다.

자동 저장은 없습니다. 중요한 변경 후에는 수동으로 저장하세요. 브라우저를 닫으면 저장하지 않은 변경 사항이 사라집니다.
